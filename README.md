# lego-odm-npc-
A Fabric 1.21.1 Minecraft mod that adds LEGO-style ODM gear NPCs with animations Titan Combat, grappling, formation and create aeronautics compatibility 
## planned Features 
- LEGO-style ODM Gear
- NPC soldiers
- Grappling hook
- Titan combat
- Formation commands
- Gas system
- Blade durability
- Multiplayer support
- Create aeronautics compatibility
